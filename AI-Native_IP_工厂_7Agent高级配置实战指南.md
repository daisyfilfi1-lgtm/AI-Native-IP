# AI-Native IP 工厂 - 7-Agent 高级配置实战指南

> 本文档详细说明每个 Agent 的高级配置项在实际业务场景中的应用价值和配置建议。

---

## 一、策略 Agent（Strategy Agent）

### 核心职责
决策"做什么内容"和"怎么投放"，24小时监控全网热点，智能评分排序选题。

### 高级配置详解

#### 1. 四维评分权重

| 维度 | 配置项 | 实战场景 | 配置建议 |
|------|--------|----------|----------|
| **流量潜力** | 30% | 预判话题热度，选择有爆发潜力的选题 | **新手IP调低到20%**：避免盲目追热点，先建立人设<br>**成熟IP保持30%**：已有粉丝基础，可适当追热点 |
| **变现关联** | 30% | 内容与产品的关联程度 | **带货IP提升到40%**：每条内容都要为转化服务<br>**知识IP保持30%**：平衡干货与变现 |
| **IP契合度** | 25% | 内容与IP人设的匹配程度 | **所有IP建议保持25%+**：这是防止"什么火拍什么"的关键<br>**垂直领域IP提升到35%**：如医疗IP必须高度专业 |
| **制作成本** | 15% | 拍摄时间、场地、人员成本 | **个人IP提升到25%**：时间有限，优先低成本高产出<br>**团队IP保持15%**：有资源支持复杂制作 |

#### 2. 竞品监控清单

**实战场景**：
- **冷启动期**：监控10-20个对标账号（同赛道、粉丝量3-10倍）
- **成长期**：监控50个账号（包括跨界账号，寻找差异化机会）
- **成熟期**：监控100+账号（全行业监控，发现新趋势）

**配置意义**：
- 系统自动抓取竞品爆款，不用人工刷抖音
- 低粉爆款（<10万粉但点赞>1万）最值得借鉴

#### 3. 选题黑名单

**实战场景**：
```
必加黑名单：
- 政治敏感词
- 竞品品牌名（避免给对手引流）
- 医疗绝对词（"最好""第一"）
- 金融诱导词（"稳赚""保本"）

行业特定：
- 教育IP：避免"保过""包就业"
- 美妆IP：避免"药妆""医疗级"
```

#### 4. 平台差异化策略

| 平台 | 特性 | 配置策略 |
|------|------|----------|
| **抖音** | 算法推荐强，完播率重要 | 偏好30-60秒，前3秒必须抓人<br>配置：时长权重高，开头钩子严格 |
| **视频号** | 社交传播强，中老年用户多 | 偏好2-3分钟，内容要有深度<br>配置：允许更长时长，情绪曲线平缓 |
| **小红书** | 图文+视频，女性用户多 | 封面图权重极高<br>配置：视觉元素提取权重高 |

#### 5. 抓取配置

**监控频率**：
- **高频（15分钟）**：热点事件期间，追时效性内容
- **中频（1小时）**：日常运营，平衡及时性与API消耗
- **低频（每天）**：稳定期IP，重质不重量

**爆款阈值**：
- **低粉账号**：1万点赞（<10万粉算低粉）
- **中粉账号**：5万点赞
- **大V账号**：10万点赞

## 二、重组 Agent（Remix Agent）

### 核心职责
竞品解构并IP化重组，保留已被验证的结构，替换为IP独家素材。

### 高级配置详解

#### 1. 解构规则

**提取要素选择**：

| 要素 | 应用场景 | 建议 |
|------|----------|------|
| **钩子模式** | 必开 | 学习爆款如何在前3秒抓注意力<br>如："90%的人都不知道..."、"我花了100万买来的教训..." |
| **情绪曲线** | 必开 | 分析爆款的情感起伏节奏<br>如：愤怒→希望→行动号召 |
| **论证结构** | 必开 | 学习如何组织论据<br>如：问题→案例→方法论→行动 |
| **视觉元素** | 可选 | 分析画面切换、特效使用<br>适合视觉型IP（如穿搭、美食） |

#### 2. 杂交策略

| 策略 | 应用场景 | 意义 |
|------|----------|------|
| **Best of Breed** | 默认推荐 | 从3-5个竞品各取最优部分<br>A的开头 + B的情绪转折 + C的结尾<br>**效果**：集众家之长，超越单一竞品 |
| **单一深度解构** | 对标爆款复制 | 深度分析1个超级爆款，完整复刻结构<br>**场景**：确认某条内容是行业标杆，要出同款 |
| **多竞品融合** | 创新内容 | 融合不同赛道的内容结构<br>**场景**：跨界创新，如把美食视频的叙事结构用在知识付费 |

#### 3. 原创度阈值

**文本重复率**：
- **严格模式：<15%**（金融、医疗等强监管行业）
- **标准模式：<25%**（大多数IP适用）
- **宽松模式：<40%**（测试内容、流量型内容）

**结构相似度**：
- **<30%**：完全看不出来借鉴了谁
- **30-40%**：行业内行能看出痕迹
- **>40%**：粉丝可能觉得"像某某"

**实战建议**：
- A级内容（人设型）：结构相似度<30%
- B级内容（流量型）：结构相似度<40%

#### 4. 强制替换规则

**100%替换清单**：
- ✅ 竞品案例 → IP自己的故事
- ✅ 竞品产品 → IP自己的产品
- ✅ 竞品数据 → IP自己的数据
- ✅ 竞品牌子 → 去掉或替换

**保留清单**：
- ✅ 行业通用数据（如"中国3亿人失眠"）
- ✅ 公共知识（如"马斯洛需求理论"）

## 三、生成 Agent（Generation Agent）

### 核心职责
转化为符合IP风格的终稿文案，风格克隆，去AI化。

### 高级配置详解

#### 1. 风格训练集

**50条S级文案选择标准**：

| 类型 | 数量 | 选择标准 |
|------|------|----------|
| **爆款视频** | 20条 | 播放量>粉丝数3倍，完整传达了IP核心价值 |
| **高转化内容** | 15条 | 带来实际询盘/成交，证明说服力 |
| **人设强化** | 10条 | 体现IP独特观点、价值观 |
| **情感共鸣** | 5条 | 评论区大量"我也是""说得太对了" |

**实战意义**：
- 训练集质量决定AI生成内容的"像不像"
- 建议每季度更新10条，保持风格与时俱进

#### 2. 风格强度

| 强度 | 应用场景 | 效果 |
|------|----------|------|
| **100%** | 完全模仿IP，甚至复制缺点 | 最像真人，但可能缺乏创新 |
| **80%** | 推荐值 | 像IP，但AI能适当发挥 |
| **60%** | 轻度模仿 | 只有IP的影子，更通用 |
| **40%** | 参考IP风格 | 主要用于测试新风格 |

#### 3. 口头禅管理

**配置方法**：
```
高频（每100字出现2-3次）：
- "你知道吗"
- "我跟你讲"

中频（每100字出现1次）：
- "说实话"
- "坦白说"

低频（偶尔出现）：
- "不瞒你说"
- "说句实在的"
```

**实战意义**：
- 口头禅是IP的"听觉指纹"
- 高频口头禅让粉丝觉得"这就是XX说话"

#### 4. 禁忌词汇

**必禁清单**：
```
太微商：
- "亲们"、"宝宝们"、"家人们"

太说教：
- "显然"、"毫无疑问"、"必须"

太AI：
- "值得注意的是"、"综上所述"、"让我们"
```

#### 5. 句长偏好

| 句型 | 比例 | 应用场景 | 效果 |
|------|------|----------|------|
| **短句<15字** | 60% | 口播视频 | 有节奏感，适合短视频 |
| **中句15-30字** | 30% | 详细解释 | 信息密度适中 |
| **长句>30字** | 10% | 复杂观点 | 展现专业深度 |

**IP类型建议**：
- **激进型IP**：短句80%，快节奏冲击
- **沉稳型IP**：短句40%，中长句为主

#### 6. 修辞偏好

**可选修辞**：
- **极端数字**："90%的人都错了"、"我花了100万"
- **身体隐喻**："死在这个坑"、"踩过的雷"
- **反问句**："你是不是也这样？"、"你觉得呢？"
- **对比**："普通人...高手..."

#### 7. 情绪曲线模板

| 模板 | 结构 | 适用内容 | 案例 |
|------|------|----------|------|
| **愤怒-希望型** | 痛点→愤怒→解决方案→希望→CTA | 解决方案类 | "深夜崩溃→我发现→你也可以" |
| **好奇-震惊型** | 悬念→揭秘→震惊→解释→CTA | 揭秘类 | "你知道为什么...真相是..." |
| **问题-解决型** | 问题→影响→方法→案例→CTA | 干货类 | "很多人不会...其实只需..." |
| **故事-升华型** | 故事→提炼→方法论→CTA | 人设类 | "当年我...这让我明白..." |

#### 8. 去AI化参数

**口语瑕疵插入**：
- **频率**：每100字插入2-3处
- **类型**："呃"、"那个"、"就是"、停顿

**时间锚点**：
- 添加具体时间："2020年3月15日凌晨"
- 添加个人记忆："那天我在北京三里屯..."

**不完美表达**：
- 打破对仗："不是说你要多努力，而是说你要..."
- 口语化转折："但是吧..."、"不过呢..."

## 四、合规 Agent（Compliance Agent）

### 核心职责
工业刹车系统，三级审查内容合规性，防止封号限流。

### 高级配置详解

#### 1. 敏感词库（三级体系）

**🔴 红线（严格阻断）**：
```
政治类：涉政敏感词、领导人姓名不当使用
色情暴力：低俗词汇、血腥描述
诈骗诱导：虚假承诺、传销术语
竞品攻击：恶意诋毁特定品牌
```
**处理**：直接拦截，无法发布

**🟡 黄线（警告提示）**：
```
广告法：最好、第一、国家级、独家
医疗绝对词：治愈、根治、100%有效
金融诱导：稳赚、保本、无风险
夸大描述：史上最、全球第一
```
**处理**：标记提醒，需人工确认后发布

**🔵 蓝线（建议替换）**：
```
"最好" → "非常不错"
"最便宜" → "性价比很高"
"绝对" → "非常"
"第一" → "领先"
```
**处理**：自动提供替换建议，可一键修改

#### 2. 平台规则适配

| 平台 | 特殊规则 | 配置重点 |
|------|----------|----------|
| **抖音** | 禁站外引流（不能说微信） | 严格检测"微信""公众号"等词<br>自动替换为"主页签名""小窗" |
| **视频号** | 禁过度营销 | 检测促销词汇密度<br>超过阈值提醒"营销感过重" |
| **小红书** | 禁最低价、禁站外导流 | 检测"最便宜""全网最低"<br>检测淘宝/天猫等外链 |

#### 3. 审核严格度

| 模式 | 应用场景 | 策略 |
|------|----------|------|
| **严格模式** | 医疗、金融、法律等强监管行业 | 宁可错杀，不放过任何风险<br>黄线词也阻断 |
| **标准模式** | 大多数IP | 红线阻断，黄线警告<br>平衡安全与效率 |
| **宽松模式** | 内部测试、非公开发布 | 仅红线阻断<br>适合快速测试内容 |

#### 4. 自动修正开关

**开启状态**：
- 检测到"最" → 自动建议改为"非常"
- 检测到"微信" → 自动建议改为"主页"

**关闭状态**：
- 仅标记不修改
- 适合人工审核严格的团队

#### 5. 行业特殊规则

**医疗行业**：
- 需额外检查资质声明
- 禁止治疗效果承诺
- 必须添加"仅供参考，不构成医疗建议"

**金融行业**：
- 需风险提示
- 禁止收益承诺
- 需资质编号

**教育行业**：
- 禁止保过承诺
- 禁止学历造假
- 需真实学员案例

## 五、视觉 Agent（Visual Agent）

### 核心职责
文案转视觉方案，B级内容数字人自动拍摄，A级内容智能分镜。

### 高级配置详解

#### 1. 数字人形象（HeyGen）

**训练视频要求**：
- **时长**：3分钟正面半身视频
- **背景**：纯色/简单背景
- **光线**：均匀自然光
- **表情**：自然微笑、适度手势

**应用场景**：
- **B级内容**：日产300条全靠数字人
- **成本对比**：$2/条 vs 实拍¥2000/条

#### 2. 声音克隆

**训练音频要求**：
- **时长**：10分钟清晰录音
- **内容**：覆盖常用词汇、语气词
- **质量**：无背景噪音，清晰人声

**参数调节**：
- **语速**：1.0（正常）、1.2（快速）、0.8（沉稳）
- **音调**：0（原声）、+10%（更亮）、-10%（更低沉）
- **情绪**：neutral（中性）、happy（积极）、serious（严肃）

#### 3. 场景库管理

**场景分类**：
```
办公场景：
- 办公室（专业感）
- 会议室（正式感）
- 咖啡厅（轻松感）

生活场景：
- 家中书房（亲近感）
- 夜景窗边（氛围感）
- 户外天台（开阔感）

特殊场景：
- 舞台（演讲感）
- 车内（随性感）
```

**实战意义**：
- 根据文案情绪自动匹配场景
- 愤怒情绪→暗色调书房
- 希望情绪→明亮的办公室

#### 4. BGM库（情绪音乐）

| 情绪 | 应用场景 | 音乐特征 |
|------|----------|----------|
| **紧张** | 钩子部分、问题提出 | 节奏快、低音重、悬疑感 |
| **愤怒** | 痛点描述 | 强烈的鼓点、厚重的贝斯 |
| **希望** | 解决方案 | 渐强、弦乐、向上的旋律 |
| **爽感** | 成功案例、结果展示 | 轻快、电子乐、胜利感 |
| **温暖** | 情感故事 | 钢琴、轻柔、舒缓 |

#### 5. 景别配比

**A级内容（人设型）**：
- 特写40%（情感细节）
- 中景40%（动作展示）
- 全景20%（环境交代）

**B级内容（流量型）**：
- 特写20%（节省制作成本）
- 中景60%（标准口播）
- 全景20%（场景切换）

**实战意义**：
- A级用特写增加感染力
- B级用中景降低数字人成本

#### 6. 情绪-景别映射

| 文案情绪 | 推荐景别 | 原因 |
|----------|----------|------|
| **愤怒/激动** | 特写 | 放大表情冲击力 |
| **讲述/叙事** | 中景 | 展示肢体语言 |
| **反思/总结** | 全景+中景 | 营造氛围+回归人物 |
| **号召/CTA** | 特写 | 建立眼神连接 |

## 六、分析 Agent（Analytics Agent）

### 核心职责
自动化数据归因与策略迭代，生成周报，反馈闭环优化。

### 高级配置详解

#### 1. 爆款定义

**播放量阈值**：
- **绝对值**：>10万播放
- **相对值**：>粉丝数×3
- **综合**：播放量>粉丝数×3 或 绝对值>10万

**互动率阈值**：
- **>10%**：超级爆款（点赞+评论+分享/播放）
- **>5%**：标准爆款
- **>3%**：潜力内容

**配置意义**：
- 统一定义什么是"好内容"
- 避免主观判断，用数据说话

#### 2. 归因权重

| 因素 | 权重 | 分析重点 |
|------|------|----------|
| **选题** | 35% | 是否选对话题？是否追热点？ |
| **开头** | 25% | 前3秒完播率如何？钩子是否有效？ |
| **内容** | 20% | 中段完播率？情绪曲线是否合理？ |
| **投放** | 10% | 是否投Dou+？投放时机？ |
| **其他** | 10% | 发布时间、封面图等 |

**实战应用**：
```
案例：视频播放量低
分析：开头权重25%，前3秒完播率<30%
结论：钩子不够吸引，优化开头
```

#### 3. 周报模板

**极简版（1页）**：
- 本周总播放量、爆款数、互动率
- 适用：高管快速了解

**标准版（5页）**：
- 数据概览+详细分析+成功案例+改进建议
- 适用：运营团队复盘

**详细版（10页）**：
- 包含每条内容分析、竞品对比、趋势预测
- 适用：深度策略制定

#### 4. 预警阈值

**CPA（单客成本）预警**：
- **阈值**：>¥100
- **触发**：自动通知"投放成本过高，需优化"

**爆款率预警**：
- **阈值**：<10%
- **触发**：提示"爆款率下降，检查选题策略"

**发布频率预警**：
- **阈值**：日更<目标数80%
- **触发**：提醒"产能不足，检查工作流"

#### 5. 数据导出

**导出格式**：
- CSV（Excel分析）
- JSON（BI系统对接）
- PDF（汇报使用）

**导出平台**：
- 飞书多维表格（推荐，团队协同）
- 钉钉
- 企业微信

## 七、7-Agent 配置组合建议

### 场景1：新IP冷启动（0-1万粉）

**策略Agent**：
- 流量潜力权重：20%（不盲目追热点）
- IP契合度权重：35%（先立人设）
- 竞品监控：10个对标账号

**生成Agent**：
- 风格强度：100%（完全模仿已有S级文案）

## 总结：配置的核心逻辑

1. **策略Agent**：决定**做什么**（选题质量）
