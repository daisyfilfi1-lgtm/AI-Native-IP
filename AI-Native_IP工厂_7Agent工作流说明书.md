## 一、产品概述

### 1.1 什么是AI-Native IP工厂？

AI-Native IP工厂是一套**工业化内容生产系统**，通过7个AI Agent的智能协作，将IP（创始人/KOL/专家）的内容产能提升**10倍**，同时保持IP独特的个人风格。

**核心价值**：
- **产能飞跃**：从日产2条提升至日产300条
- **风格一致**：AI生成内容与IP本人撰写相似度>85%
- **成本降低**：单条内容成本从¥1,333降至¥48
- **时间释放**：IP每日投入从8小时降至45分钟

## 二、7-Agent详解

### Agent 1：策略Agent（选题智囊团）

#### 核心作用
**解决"拍什么"的问题**

- **全网监控**：24小时抓取抖音/视频号/小红书低粉爆款（<10万粉但点赞>1万）
- **智能评分**：从流量潜力、变现关联、IP契合度、制作成本四个维度自动评分
- **预测性选题**：基于历史数据预测未来7天可能爆发的话题

#### 关键创新
- **四维评分模型**：不是凭感觉选题，而是数据驱动决策
- **竞品拆解自动**：自动分析爆款结构，识别可借鉴要素

#### 客户可配置部分

| 配置项 | 说明 | 示例 |
|--------|------|------|
| **评分权重** | 调节四个维度的权重占比 | 流量30% + 变现30% + IP契合25% + 成本15% |
| **竞品监控清单** | 指定监控哪些对标账号 | 添加20个竞品账号，系统自动追踪 |
| **选题黑名单** | 设置绝不触碰的话题 | 政治/宗教/特定竞品名称 |
| **平台策略** | 不同平台的差异化规则 | 抖音偏好30-60秒，小红书重封面图 |

**客户价值**：告别"拍脑袋选题"，每个选题都有数据支撑

---

### Agent 2：记忆Agent（IP数字大脑）

#### 核心作用
**解决"用什么素材"的问题**

- **素材数字化**：将IP过往所有内容（视频/音频/文字）转化为可检索的数字资产
- **智能标签化**：自动识别情绪标签（愤怒/焦虑/爽感）、场景标签（深夜/办公室）、认知标签（反常识/揭秘）
- **语义检索**：支持模糊查询，如"找一个深夜崩溃但最终翻盘的故事"

#### 关键创新
- **向量数据库**：不是关键词搜索，而是语义理解（找"失眠"也能找到"辗转反侧"）
- **知识图谱**：建立故事与观点的关联（如"2020破产夜"故事引出"反脆弱财务"观点）
- **新鲜度控制**：自动标记高频使用素材，防止"讲烂了"

#### 客户可配置部分

| 配置项 | 说明 | 示例 |
|--------|------|------|
| **标签体系** | 自定义分类维度 | 情绪标签（愤怒/焦虑/希望）、场景标签（办公室/车内/舞台） |
| **素材分级** | 设置使用频次限制 | 核心素材（至暗时刻）限用3次，常规素材限用10次 |
| **检索策略** | 调节素材推荐逻辑 | 优先新素材（新鲜度权重20%）或优先经典素材 |
| **自动打标复核** | AI打标后人工确认 | 确认"2020破产夜"的情绪是"愤怒"还是"绝望" |

**客户价值**：IP的每个故事都被永久保存、智能调用，永不遗忘

---

### Agent 3：重组Agent（创意重组师）

#### 核心作用
**解决"怎么写结构"的问题**

- **竞品解构**：分析爆款视频的"黄金3秒""情绪转折点""结尾钩子"
- **合法借鉴**：保留已被验证的结构，替换内容为IP独家素材
- **观点升维**：将"个人经验"升级为"行业规律/认知科学"

#### 关键创新
- **结构杂交**：从3-5个竞品各取最优部分（A的开头+B的冲突+C的结尾）
- **DNA替换**：100%替换案例为IP素材，确保原创性
- **原创度保障**：查重率<25%，结构相似度<40%

#### 客户可配置部分

| 配置项 | 说明 | 示例 |
|--------|------|------|
| **解构规则** | 提取竞品的哪些要素 | 提取钩子模式、情绪曲线、论证结构 |
| **杂交策略** | 如何组合多个竞品 | best_of_breed（每个节点取最优） |
| **原创度阈值** | 安全线设置 | 文本重复率<25%，结构相似度<40% |
| **强制替换规则** | 哪些必须替换 | 竞品案例100%替换为IP素材 |

**客户价值**：合法"站在巨人肩膀上"，借鉴结构但内容完全原创

---

### Agent 4：生成Agent（风格克隆器）★核心

#### 核心作用
**解决"写得像不像我"的问题**

- **风格克隆**：基于IP 50条S级文案Fine-tune专属模型
- **语言指纹**：植入口头禅、句长偏好、修辞习惯
- **情绪工程**：按"W型情绪曲线"排布文案，每15秒一次情绪波动
- **去AI化**：插入口语瑕疵（"呃""那个"），打破对仗工整，增加真实感

#### 关键创新
- **双模型架构**：Base Model（风格模仿）+ RAG（实时素材注入）
- **风格强度调节**：0-100%可调，越强越像IP，越弱越通用
- **反AI检测**：自动规避平台AI内容识别，避免限流

#### 客户可配置部分

| 配置项 | 说明 | 示例 |
|--------|------|------|
| **风格训练集** | 精选50条S级文案 | 上传过往爆款文案，AI学习风格 |
| **口头禅管理** | 标志性短语 | "你知道吗""我跟你讲""说实话"，设置出现频率 |
| **句长偏好** | 长短句比例 | 短句60%（<15字）+ 中句30% + 长句10% |
| **修辞偏好** | 喜欢的表达手法 | 极端数字（90%/100万）、身体隐喻（"死在这个坑"）、反问句 |
| **禁忌词汇** | 绝对不用 | "亲们""宝宝们"（太微商）、"显然"（太说教） |
| **风格强度** | 像不像的调节 | 80%强度（推荐）：既像IP又有新意 |
| **情绪曲线模板** | 文案节奏 | 愤怒-希望型（解决方案类）、好奇-震惊型（揭秘类） |
| **去AI化参数** | 真实感调节 | 每100字插入2处口语瑕疵（"呃""那个"） |
| **变现植入规则** | 产品自然出现 | 产品只在"解决方案"段落出现，先讲痛点再出产品 |

**客户价值**：AI写的文案，粉丝分不清是真人还是AI，保持IP人格一致性

### Agent 5：合规Agent（法务风控官）

#### 核心作用
**解决"安不安全"的问题**

- **三级审查**：
  - 🔴 红线（严格阻断）：政治敏感、绝对化用语、虚假宣传
  - 🟡 黄线（警告提示）：轻微违规、需人工确认
  - 🟢 绿线（自动通过）：安全内容
- **平台适配**：抖音/视频号/小红书不同规则自动适配
- **自动修正**：检测违规后自动生成替换建议

#### 关键创新
- **分平台规则库**：抖音禁"微信"，小红书禁"最便宜"，自动识别
- **实时敏感词库**：持续更新广告法禁用词、平台新规
- **归因溯源**：每条文案标记使用的素材来源，合规问题可追溯

#### 客户可配置部分

| 配置项 | 说明 | 示例 |
|--------|------|------|
| **敏感词库（三级）** | 自定义红线/黄线/蓝线 | 红线：政治/竞品名；黄线：医疗绝对词；蓝线：建议替换词 |
| **平台规则** | 不同平台不同策略 | 抖音：禁站外引流；视频号：禁过度营销 |
| **审核严格度** | 宁可错杀 vs 宁可放过 | 严格模式（保守）或宽松模式（激进） |
| **自动修正开关** | 是否自动提供修改建议 | 开启："最"→"非常"；关闭：仅标记不修改 |
| **行业特殊规则** | 医疗/金融/教育强监管 | 医疗内容需额外资质检查 |

**客户价值**：告别"发出去就限流"的焦虑，系统自动做第一道风控

### Agent 6：视觉Agent（拍摄执行导演）

#### 核心作用
**解决"怎么拍/怎么出片"的问题**

**B级内容（流量型）**：
- **数字人自动拍摄**：HeyGen生成IP形象口播视频，2小时出片
- **成本**：$2/条 vs 实拍¥2,000/条

**A级内容（人设型）**：
- **智能分镜脚本**：根据文案情绪自动生成景别、运镜、场景
- **拍摄清单**：自动输出设备、道具、灯光布置图

#### 关键创新
- **情绪-景别映射**：愤怒情绪→特写（冲击力），叙事→中景（动作）
- **场景智能推荐**：基于文案场景标签匹配历史拍摄场地
- **BGM情绪库**：按"紧张/温暖/励志"分类，自动匹配文案情绪

#### 客户可配置部分

| 配置项 | 说明 | 示例 |
|--------|------|------|
| **数字人形象** | 上传训练视频克隆形象 | 3分钟正面半身视频，生成专属数字人 |
| **声音克隆** | 上传音频克隆音色 | 上传10分钟清晰录音，克隆声音 |
| **场景库** | 常用拍摄场地管理 | 办公室/咖啡厅/夜景窗边，附照片和光线条件 |
| **BGM库** | 情绪音乐分类 | 紧张/愤怒/希望/爽感，每类3-5首备选 |
| **景别配比** | A级/B级差异 | A级：特写40%+中景40%；B级：特写20%+中景60% |
| **数字人参数** | 语速/音调/情绪 | 语速1.0（正常），音调0，基础情绪neutral |

**客户价值**：B级内容全自动出片，A级内容有专业分镜指导，降低对导演依赖

### Agent 7：分析Agent（数据分析师）

#### 核心作用
**解决"为什么好/为什么差"的问题**

- **实时监测**：发布后每2小时抓取播放量/完播率/互动率/客资量
- **智能归因**：自动分析爆款成功因素（选题/开头/投放）或失败原因
- **自动周报**：每周五生成数据复盘报告，自动提取可复用组件入库

#### 关键创新
- **爆款组件提取**：自动拆解S级内容的"黄金3秒模板""金句""视觉包装"
- **反馈闭环**：爆款特征→优化选题算法，失败归因→调整重组策略
- **预测准确性**：对比预测数据vs实际数据，持续校准系统

#### 客户可配置部分

| 配置项 | 说明 | 示例 |
|--------|------|------|
| **爆款定义** | 什么算爆款 | 播放量>粉丝数×3，或绝对值>10万 |
| **归因权重** | 什么因素最重要 | 选题35% + 开头25% + 内容20% + 投放10% |
| **周报模板** | 报告详细程度 | 标准版（5页）/详细版（10页）/极简版（1页） |
| **数据导出** | 对接BI系统 | 自动导出CSV到飞书/钉钉/企业微信 |
| **预警阈值** | 异常自动提醒 | CPA>100元时告警，爆款率<10%时告警 |

**客户价值**：数据驱动决策，系统越用越懂IP，爆款率持续提升

## 三、客户最关心的3个问题

### Q1：AI写的会不会太假，粉丝能看出来吗？

**回答**：
- **风格克隆技术**：我们基于您50条S级文案Fine-tune专属模型，AI学会您的口头禅、句长、修辞习惯
- **去AI化处理**：自动插入"呃""那个"等口语瑕疵，打破对仗工整，增加具体时间锚点（"2020年3月15日凌晨"）
- **盲测验证**：87%的粉丝分不清是真人还是AI撰写
- **您有终审权**：所有内容必须经过您的"通过"按钮才会发布

**您可配置**：风格强度（0-100%）、口头禅库、禁忌词汇表、去AI化参数

---

### Q2：我的独家故事会不会被用烂，或者泄露？

**回答**：
- **使用频次限制**：核心素材（如至暗时刻）设置使用上限（如3次），用完后系统自动提醒"该补采新故事"
- **新鲜度权重**：新素材优先推荐，老素材逐渐降权
- **数据隔离**：您的素材库独立存储，多IP物理隔离
- **溯源管理**：每条生成内容标记使用了您的哪些素材，全程可追溯

**您可配置**：素材分级（核心/常规/消耗）、使用次数限制、新鲜度权重

---

### Q3：不合规的内容会不会发出去导致封号？

**回答**：
- **三级审核机制**：
  - 🔴 红线（严格阻断）：政治敏感、绝对化用语→直接拦截，无法发布
  - 🟡 黄线（警告提示）：轻微违规→标记提醒，需您确认后才可发布
  - 🟢 绿线（自动通过）：安全内容→进入发布队列
- **分平台规则**：抖音/视频号/小红书不同规则自动适配
- **自动修正**：检测到"最"自动建议改为"非常"

**您可配置**：敏感词库（三级）、平台规则、审核严格度（宁可错杀/宁可放过）

```

    ├─ 09:15 一键启动工作流，AI自动生成内容
    ├─ 11:00 收到"待审核"提醒，查看3-5条AI生成内容
    ├─ 11:30 点击"通过"或"修改"（修改后AI自动优化）
    └─ 12:00 内容自动发布，分析Agent开始监测数据

第5步：周报复盘（每周五）
    ↓ 查看自动生成的数据复盘报告，调整下周策略
```
