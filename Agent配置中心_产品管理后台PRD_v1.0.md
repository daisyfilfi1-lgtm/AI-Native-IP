## 一、产品概述

### 1.1 产品定位
Agent配置中心是AI-Native IP工厂的管理后台，提供7大Agent的可视化配置能力，让非技术人员（IP主理人、内容总监）也能自定义AI内容生产规则。

### 1.2 目标用户
| 角色 | 主要职责 | 使用模块 |
|------|---------|---------|
| 超级管理员 | 系统级配置、权限分配 | 全部 |
| IP主理人 | 定义个人风格、审核内容 | 记忆Agent、生成Agent、视觉Agent |
| 内容总监 | 把控内容策略、数据分析 | 策略Agent、重组Agent、分析Agent |
| 法务审核员 | 合规规则配置 | 合规Agent |
| 运营专员 | 上传素材、日常监控 | 记忆Agent、系统监控 |

### 1.3 核心价值
- **低门槛配置**：无需代码，可视化配置7大Agent
- **实时预览**：配置后立即看到效果（如风格强度调节）
- **版本管理**：支持配置回滚，防误操作
- **权限隔离**：敏感配置（如合规规则）需特殊权限

## 模块4-7：其他Agent配置（简要）

### 3.4 重组Agent配置
**核心配置项**：
- 解构规则：提取哪些要素（钩子/情绪/结构）
- 杂交策略：best_of_breed / random_combine / weighted
- 原创度阈值：text_similarity < 0.25, structure_similarity < 0.40
- 强制替换规则：必须替换所有竞品案例为IP素材

### 3.5 合规Agent配置
**核心配置项**：
- 敏感词库（三级：红/黄/蓝）
- 平台规则（抖音/视频号/小红书差异）
- 自动修正开关
- 审核流配置（自动通过/人工复核/阻断）

### 3.6 视觉Agent配置
**核心配置项**：
- 数字人形象管理（HeyGen avatar_id/voice_id）
- 场景库管理（照片/光线/可用时间）
- BGM情绪库管理（曲目/BPM/版权）
- 景别配比（A级/B级差异）

### 3.7 分析Agent配置
**核心配置项**：
- 爆款定义阈值（播放量倍数/互动率）
- 归因权重（选题/开头/内容/投放占比）
- 周报模板选择（标准/详细/极简）
- 数据导出配置

| 运营专员 | 记忆Agent（查看/上传）、系统监控 | 仅查看，不可修改规则 |
