## 🚀 部署命令

### Linux/Mac
```bash
# 一键部署
bash scripts/deploy_v3.sh
```

### Windows
```powershell
# 一键部署
.\scripts\deploy_v3.ps1
```

### 手动部署
```bash
# 1. 备份
cp -r backend/app/services backend/app/services.backup.$(date +%Y%m%d)

# 2. 复制新文件（假设新代码在 new_code/ 目录）
cp -r new_code/datasource backend/app/services/
cp new_code/*.py backend/app/services/

# 3. 修改 main.py 添加路由注册
# （参考上线部署指南）

# 4. 启动服务
cd backend
python -m uvicorn app.main:app --reload
```

## 🆘 紧急回滚

如果上线后出现问题：

```bash
# 1. 停止服务
pkill -f uvicorn

# 2. 恢复备份
rm -rf backend/app/services/datasource
cp -r backend/app/services.backup.XXXX backend/app/services/datasource

# 3. 注释掉 main.py 中的新路由

# 4. 重启
```
