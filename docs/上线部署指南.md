## 🚀 快速部署步骤

### Step 1: 备份现有代码
```bash
cd /path/to/your/project
cp -r backend/app/services backend/app/services.backup.$(date +%Y%m%d)
```

### Step 2: 复制新文件
```bash
# 复制整个 datasource 目录
cp -r backend/app/services/datasource backend/app/services/

# 复制新服务文件
cp backend/app/services/keyword_synonyms.py backend/app/services/
cp backend/app/services/enhanced_topic_matcher.py backend/app/services/
cp backend/app/services/topic_rewrite_service.py backend/app/services/
cp backend/app/services/topic_recommendation_v3.py backend/app/services/
cp backend/app/services/builtin_topic_repository.py backend/app/services/

# 复制新路由
cp backend/app/routers/topic_recommendation_v2.py backend/app/routers/
```

### Step 3: 注册路由（修改 main.py）
在 `backend/app/main.py` 中添加：

```python
# 导入新路由
from app.routers import topic_recommendation_v2

# 注册路由（在原有路由之后）
app.include_router(
    topic_recommendation_v2.router,
    prefix="/api/v1",
    tags=["topic-recommendation-v2"]
)
```

### Step 4: 配置环境变量（最低配置）

#### 方案A：零成本启动（推荐先验证）
```bash
# 只使用内置库，无需外部API
# 无需配置任何环境变量，直接启动
```

#### 方案B：低成本生产（推荐）
```bash
# 配置顺为数据（10元/月，小红书/抖音数据）
export SHUNWEI_API_KEY="your_api_key_here"

# 配置DailyHot（免费备用）
export DAILYHOT_API_URL="https://api-hot.imsyy.top"
```

#### 方案C：已有TIKHUB
```bash
# 保留原有TIKHUB配置
export TIKHUB_API_KEY="your_existing_key"
export TIKHUB_BASE_URL="https://api.tikhub.io"
```
