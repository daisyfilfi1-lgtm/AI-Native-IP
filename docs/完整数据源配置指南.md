## 🔧 快速配置步骤

### Step 1: 基础配置（必选）
```env
# 设置基本环境变量
ENV=production
```

### Step 2: 选择数据源方案

**如果预算有限，选免费方案**:
```env
DAILYHOT_API_URL=https://api-hot.imsyy.top
```

**如果有少量预算，选低成本方案**:
1. 注册顺为数据: https://api.itapi.cn/ （10元/月）
2. 获取API Key并配置:
```env
SHUNWEI_API_KEY=your_key_here
```

**如果需要更多数据，加购QQ来客源**:
1. 注册QQ来客源: https://qqlykm.cn/ （10元/3000次）
2. 配置:
```env
SHUNWEI_API_KEY=your_shunwei_key
QQLYKM_API_KEY=your_qqlykm_key
```

### Step 3: 验证配置
```python
from app.services.datasource import get_datasource_manager_v2

manager = get_datasource_manager_v2()

# 查看可用数据源
available = manager.list_available_sources()
print(f"可用数据源: {available}")

# 查看数据源状态
all_sources = manager.list_sources()
for s in all_sources:
    print(f"{s['name']}: {s['status']} - 可用: {s['enabled']}")
```

## 🛠️ 故障排查

### 问题1: 所有数据源都不可用
**检查**: 
```python
manager = get_datasource_manager_v2()
sources = manager.list_sources()
for s in sources:
    print(f"{s['source_id']}: available={s.get('health', {}).get('available')}")
```

**解决**:
- 确保至少配置了 `DAILYHOT_API_URL` 或 `SHUNWEI_API_KEY`
- 检查网络连接
- 查看日志确认具体错误

### 问题2: 小红书数据少
**检查**:
```python
topics = await manager.fetch_from_platform("xiaohongshu", ip_profile, 10)
print(f"获取到 {len(topics)} 条小红书数据")
```

**解决**:
- 配置 `SHUNWEI_API_KEY` 获取更大小红书数据
- 顺为数据的小红书热点质量较高

### 问题3: API调用超限
**检查**:
- 顺为数据: 3000次/天
- QQ来客源: 按购买的次数

**解决**:
- 启用缓存: 系统自动缓存，减少API调用
- 增加数据源: 配置多个源分担压力
- 升级套餐: 购买更高额度
