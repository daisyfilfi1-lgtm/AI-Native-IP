## 🚀 快速使用

### 方式1：一行代码获取数据
```python
from app.services.datasource import fetch_topics_v2

# 使用智能策略自动选择最佳数据源
topics = await fetch_topics_v2(ip_profile, limit=12, strategy="smart")
```

### 方式2：获取特定平台数据
```python
from app.services.datasource import get_datasource_manager_v2

manager = get_datasource_manager_v2()

# 获取小红书数据
xhs_topics = await manager.fetch_from_platform("xiaohongshu", ip_profile, 10)

# 获取抖音数据
dy_topics = await manager.fetch_from_platform("douyin", ip_profile, 10)

# 获取多平台数据
all_data = await manager.fetch_all_platforms(ip_profile, platforms=["xiaohongshu", "douyin"])
```

### 方式3：使用不同策略
```python
# 只使用免费源
topics = await manager.fetch_with_strategy(ip_profile, limit=12, strategy="free_only")

# 优先付费源
topics = await manager.fetch_with_strategy(ip_profile, limit=12, strategy="paid_first")

# 按平台聚合
topics = await manager.fetch_with_strategy(ip_profile, limit=12, strategy="platform")
```

## 📊 数据源获取指南

### 小红书数据（优先）

**推荐：顺为数据（10元/月）**
1. 访问 https://api.itapi.cn/
2. 注册账号
3. 购买"小红书热点"套餐（10元/月）
4. 获取API Key并配置

**备选：DailyHot（免费）**
1. 使用公共实例: https://api-hot.imsyy.top
2. 或自部署: https://github.com/imsyy/DailyHotApi

### 抖音数据

**推荐：顺为数据（10元/月）**
- 同上，支持抖音热点

**备选：DailyHot + 免费API**
- 配置 DailyHot API URL
- 系统自动聚合多个免费源

### 微博数据

**免费可用**
- 微博热搜免费API（自动）
- DailyHot（自动）
- VVhan（自动）
