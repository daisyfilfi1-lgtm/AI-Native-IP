# 数据源快速参考卡片

## 🎯 推荐配置（复制即用）

### 方案A：免费起步（零成本）
```env
DAILYHOT_API_URL=https://api-hot.imsyy.top
```

### 方案B：低成本（推荐，约20元/月）
```env
# 顺为数据 - 小红书/抖音热点
SHUNWEI_API_KEY=your_key_here

# QQ来客源 - 多平台补充  
QQLYKM_API_KEY=your_key_here

# DailyHot备用
DAILYHOT_API_URL=https://api-hot.imsyy.top
```

## 🔧 策略选择

| 策略 | 用途 | 代码 |
|------|------|------|
| smart | 智能选择（推荐） | `strategy="smart"` |
| free_only | 只用免费源 | `strategy="free_only"` |
| paid_first | 优先付费源 | `strategy="paid_first"` |
| platform | 按平台聚合 | `strategy="platform"` |

## ⚡ 故障排查

### 问题：没有数据返回
```python
# 检查可用数据源
manager = get_datasource_manager_v2()
print(manager.list_available_sources())
```

### 问题：数据质量差
- 配置 `SHUNWEI_API_KEY` 获取更大小红书数据
- 使用 `strategy="paid_first"` 优先高质量源

### 问题：API超限
- 启用缓存（自动）
- 配置多个数据源分担压力
