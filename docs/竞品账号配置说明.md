## 🚀 快速配置

### 1. 创建数据库表

```bash
# 连接到PostgreSQL
psql -U postgres -d ip_factory -f backend/scripts/create_competitor_tables.sql
```

### 2. 插入竞品账号配置

```bash
psql -U postgres -d ip_factory -f backend/scripts/setup_competitors_from_analysis.sql
```

### 3. 插入示例视频数据（用于测试）

```bash
psql -U postgres -d ip_factory -f backend/scripts/seed_competitor_videos.sql
```

## 📊 数据验证

配置完成后，验证数据是否正确插入：

```sql
-- 查看竞品账号
SELECT competitor_id, name, followers_display, LEFT(notes, 30) as notes
FROM competitor_accounts 
WHERE ip_id = 'xiaomin';

-- 查看视频数据
SELECT 
    c.name,
    v.title,
    v.play_count,
    v.content_type,
    v.content_structure->>'angle' as angle
FROM competitor_videos v
JOIN competitor_accounts c ON v.competitor_id = c.competitor_id
WHERE c.ip_id = 'xiaomin'
ORDER BY v.play_count DESC
LIMIT 10;
```

## 🔄 自动抓取竞品视频

### 使用TIKHUB API实时抓取

```bash
# 设置环境变量
export TIKHUB_API_KEY="your_api_key"
export DATABASE_URL="postgresql://postgres:postgres@localhost:5432/ip_factory"

# 运行抓取脚本
python backend/scripts/fetch_competitor_videos.py --ip-id xiaomin --limit 10
```

### 定时任务（推荐）

添加cron任务，每天自动更新竞品数据：

```bash
# 编辑crontab
crontab -e

# 添加定时任务（每天凌晨2点抓取）
0 2 * * * cd /path/to/backend && python scripts/fetch_competitor_videos.py --ip-id xiaomin --limit 10 >> /var/log/competitor_fetch.log 2>&1
```
