# 基于IP的匹配爆款选题 - 优化方案 v2.0

## 一、核心思路转变

### 旧模式（问题根源）
```
通用热榜 → 关键词过滤 → IP改写 → 推荐
```
- 抖音热榜：国足、明星八卦、社会新闻...
- 过滤后几乎为空
- 强制改写导致质量差

### 新模式（推荐）
```
IP定位分析 → 定向爬取/搜索 → 语义匹配 → 四维评分 → 推荐
```
- 先理解IP需要什么内容
- 主动定向获取相关内容
- 多维度评估匹配度

### 3.3 内容矩阵智能分配

```python
def allocate_content_matrix(topics: List[Dict], ip_id: str) -> List[Dict]:
    """
    按4-3-2-1内容矩阵分配选题
    
    目标比例：
    - 40% 搞钱方法论 (money)
    - 30% 情感共情 (emotion)  
    - 20% 技术展示 (skill)
    - 10% 美好生活 (life)
    """
    matrix_distribution = {"money": 0.4, "emotion": 0.3, "skill": 0.2, "life": 0.1}
    
    # 为每个话题分类
    for topic in topics:
        topic["content_type"] = classify_content_type(topic["title"])
    
    # 按比例选择
    result = []
    for content_type, ratio in matrix_distribution.items():
        count = int(len(topics) * ratio)
        type_topics = [t for t in topics if t["content_type"] == content_type]
        result.extend(type_topics[:count])
    
    return result
```

## 四、业务层优化

### 4.1 推荐流程重构

```python
# 新的推荐流程（伪代码）
async def recommend_topics_v2(db, ip_id: str, limit: int = 12) -> List[Dict]:
    # Step 1: 获取IP画像
    ip_profile = get_ip_profile(db, ip_id)
    
    # Step 2: 多源数据获取（并行）
    sources = await asyncio.gather(
        fetch_competitor_videos(ip_id),      # P0: 竞品监控
        search_by_keywords(ip_id),           # P1: 关键词搜索
        fetch_xhs_topic_notes(ip_id),        # P2: 小红书话题
        fetch_low_fan_explosion(ip_id),      # P3: 低粉爆款
    )
    
    # Step 3: 合并去重
    all_topics = merge_and_deduplicate(sources)
    
    # Step 4: 语义匹配评分
    scored_topics = []
    for topic in all_topics:
        match_scores = matcher.compute_match_score(ip_profile, topic)
        topic["match_score"] = match_scores["overall"]
        topic["match_details"] = match_scores
        scored_topics.append(topic)
    
    # Step 5: 四维评分
    for topic in scored_topics:
        topic["four_dim_score"] = calculate_four_dim_score(topic, ip_profile)
    
    # Step 6: 内容矩阵分配
    final_topics = allocate_content_matrix(scored_topics, ip_id)
    
    # Step 7: 兜底机制
    if len(final_topics) < limit:
        builtin_topics = get_builtin_topics(ip_id, limit - len(final_topics))
        final_topics.extend(builtin_topics)
    
    return final_topics[:limit]
```
