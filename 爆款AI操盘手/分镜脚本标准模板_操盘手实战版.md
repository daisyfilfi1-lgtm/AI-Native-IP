# 分镜脚本标准模板（操盘手实战版）
**使用阶段**：Stage 5 - 可视化脚本制作  
**使用人**：内容策划（初稿）→ 拍摄导演（细化+统筹）→ IP主理人（确认）  
**耗时**：B级15-20分钟/条，A级45-60分钟/条  
**打印份数**：导演1份+摄像1份+IP1份（现场使用）  
**存档要求**：拍摄完成后与原始素材一同存档，标注实际拍摄与分镜差异

### 景别标准化说明（示意图）

```
远景：人物占画面<1/4，展现场景环境
全景：人物全身，头顶/脚底留空
中景：人物膝盖以上，适合叙事
近景：人物胸部以上，适合情感表达
特写：人物肩部以上或局部（手/眼），适合强调
大特写：眼睛/嘴唇等局部，适合极致情绪
```

**使用频率建议**：
- **A级内容**：特写30% + 近景40% + 中景20% + 全景10%
- **B级内容**：特写20% + 近景30% + 中景40% + 全景10%
- **C级内容**：以中景/近景为主，快速拍摄

## 附录：快速参考卡（可单独打印）

### 景别速查
- **特写**：强调、情绪高潮、金句
- **近景**：叙事、讲解、情感表达
- **中景**：动作展示、环境互动
- **全景**：环境展示、大动作

### 情绪表演提示
- **愤怒**：语速快、音量高、眼神锐利、手势有力
- **悲伤**：语速慢、音量低、眼神回避、身体蜷缩
- **兴奋**：语速快、音调高、眼神发光、身体前倾
- **神秘**：语速慢、音量低、眼神聚焦、身体靠近镜头

### 常见问题急救
- **忘词**：看提词器/助理举大字报/切特写掩盖
- **笑场**：休息5分钟/聊别的/先拍别的镜头
- **虚焦**：检查对焦模式/缩小光圈/固定机位
- **过曝**：降ISO/缩光圈/上ND滤镜
