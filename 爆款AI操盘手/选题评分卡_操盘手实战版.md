# 选题评分卡（操盘手实战版）
**使用阶段**：Stage 3 - 选题决策与流量预测  
**使用频率**：每日晨会前/每周选题会  
**决策时效**：评分后24小时内必须决定执行或废弃  
**存档要求**：所有评分≥3.0的选题进入选题库，评分卡需关联至选题数据库

## 二、四维评估矩阵（核心评分区）

### 2.1 维度A：流量潜力（权重30%）
*评估该选题在平台算法中的先天传播基因*

| 评估项 | 评分标准 | 得分 |
|-------|---------|------|
| **A1-需求验证** | 5分：近期同选题有3条以上百万级爆款<br>4分：近期有1-2条五十万级爆款<br>3分：近期有低粉爆款（<1万粉点赞>1万）<br>2分：历史有过爆款但时间久远（>3个月）<br>1分：无验证，纯原创假设 | /5 |
| **A2-情绪强度** | 5分：触发强烈情绪（愤怒/焦虑/爽感/好奇）<br>4分：解决具体痛点（有明确"怎么办"需求）<br>3分：提供信息增量（"原来如此"型）<br>2分：轻度共鸣（"确实是这样"型）<br>1分：纯理性科普，无情绪波动 | /5 |
| **A3-争议性/话题性** | 5分：天然存在正反方（评论区必吵架）<br>4分：反常识观点（挑战固有认知）<br>3分：揭秘/内幕（信息差明显）<br>2分：经验分享（温和有用）<br>1分：说明书式内容（无讨论空间） | /5 |
| **A4-平台适配** | 5分：完美契合当前平台热点（热点宝TOP100）<br>4分：契合平台长期推荐方向（如抖音的"有用性"）<br>3分：中性内容，不违规则<br>2分：有轻微违规风险（需弱化处理）<br>1分：平台近期打压类目（如炫富/焦虑营销） | /5 |
| **维度A小计** | (A1+A2+A3+A4)/4 × 30% = _______分 | |

**数据来源工具**：灰豚数据（低粉爆款）/ 热点宝（趋势）/ 巨量算数（关键词热度）

### 2.2 维度B：变现关联（权重30%）
*评估该选题与后端产品/服务的变现链路清晰度*

| 评估项 | 评分标准 | 得分 |
|-------|---------|------|
| **B1-需求匹配** | 5分：直接解决产品目标客户的核心痛点（买前必看）<br>4分：解决目标客户的周边痛点（建立信任）<br>3分：解决泛人群痛点（可筛选出目标客户）<br>2分：与产品弱相关（仅建立专业度）<br>1分：与产品无关（纯流量型） | /5 |
| **B2-植入自然度** | 5分：产品就是解决方案的核心（不突兀）<br>4分：产品是解决方案的重要环节（自然过渡）<br>3分：产品作为案例/工具出现（软性植入）<br>2分：结尾硬广植入（影响体验）<br>1分：无法植入产品（纯内容） | /5 |
| **B3-客资转化率** | 5分：评论区必问"怎么买/怎么联系"（高意向）<br>4分：会咨询"多少钱/有用吗"（中意向）<br>3分：会求"教程/资料"（可培育）<br>2分：纯点赞收藏（意向模糊）<br>1分：看完即走（无转化动作） | /5 |
| **B4-LTV贡献** | 5分：吸引高客单价客户（LTV>1万）<br>4分：吸引中客单价客户（LTV 3000-1万）<br>3分：吸引低客单价但复购率高客户<br>2分：吸引一次性购买客户<br>1分：吸引白嫖/低价敏感客户 | /5 |
| **维度B小计** | (B1+B2+B3+B4)/4 × 30% = _______分 | |

### 2.3 维度C：IP契合度（权重25%）
*评估IP是否有独家内容支撑，避免同质化*

| 评估项 | 评分标准 | 得分 |
|-------|---------|------|
| **C1-案例独占性** | 5分：使用IP独家经历/数据（外界无法复制）<br>4分：使用IP客户案例（经授权）<br>3分：使用IP方法论（有理论体系）<br>2分：使用行业通用案例（但IP有独特解读）<br>1分：纯网络素材整合（无IP烙印） | /5 |
| **C2-观点锐度** | 5分：IP有极端/鲜明观点（可能得罪人）<br>4分：IP有反共识但合理的观点<br>3分：IP有细化/升级的主流观点<br>2分：IP重复行业共识<br>1分：IP对此话题无明确立场 | /5 |
| **C3-表达意愿** | 5分：IP对此话题极度兴奋（主动想讲）<br>4分：IP有表达欲且准备充分<br>3分：IP愿意配合但需引导<br>2分：IP兴趣一般（为配合而讲）<br>1分：IP抵触或不适合讲（人设冲突） | /5 |
| **C4-专业深度** | 5分：IP有10年以上深耕/一手数据<br>4分：IP有实战案例支撑<br>3分：IP有系统学习+思考<br>2分：IP了解但不深入<br>1分：IP需要临时补知识（风险高） | /5 |

### 2.4 维度D：制作成本（权重15%）
*评估团队资源占用与机会成本*

| 评估项 | 评分标准 | 得分 |
|-------|---------|------|
| **D1-IP时间成本** | 5分：利用现有素材/口播即可完成（<30分钟）<br>4分：简单拍摄（1小时内）<br>3分：中等复杂度（需2-3小时）<br>2分：高复杂度（需半天以上）<br>1分：极高复杂度（需外出/多人/特殊场地） | /5 |
| **D2-制作难度** | 5分：单人口播/混剪（剪辑<2小时）<br>4分：需简单布景/道具（剪辑4小时）<br>3分：需多机位/简单剧情（剪辑1天）<br>2分：需专业拍摄/演员（剪辑2天+）<br>1分：需外包制作（不可控因素多） | /5 |
| **D3-时效性** | 5分：常青话题（3年后仍有效）<br>4分：年度周期话题（每季度可发）<br>3分：月度热点（本月内有效）<br>2分：周度热点（本周内必须发）<br>1分：瞬时热点（48小时内必须发） | /5 |
| **D4-试错成本** | 5分：数据不好可隐藏重剪（无损失）<br>4分：数据不好可部分修改（损失30%工时）<br>3分：需重拍（损失50%工时）<br>2分：涉及特殊资源（如嘉宾档期，损失80%）<br>1分：不可重做（如直播切片/事件营销） | /5 |

### 3.2 决策矩阵

| 分值区间 | 决策 | 执行策略 | 资源分配 |
|---------|------|---------|---------|
| **≥4.0分** | 🟢 **立即执行** | S级项目，优先排期，A级制作标准 | IP时间优先保障，投放预算倾斜 |
| **3.5-3.9分** | 🟡 **排期执行** | 优质项目，B级制作标准 | 正常排期，可尝试AB测试开头 |
| **3.0-3.4分** | 🟠 **改造后执行** | 有潜力但需优化（改选题/改角度/降成本） | 降级为C级快速测试，或与其他选题合并 |
| **<3.0分** | 🔴 **废弃/冷冻** | 暂时不做，进入"选题冷冻库"（每月复检） | 不分配资源 |
